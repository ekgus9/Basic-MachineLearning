10-1 Unsupervised Learning의 이해
-Unsupervised Learning 정의
주어진 데이터에 라벨 없고 피쳐만 존재하는 샘플 ex. 아웃라이어 : 평소와 다른 무언가 탐지 , 해일

-Clustering 정의
라벨 데이터 제공되지 않음 -> 출력값 도출
단순하게 거리가 가까운 샘플들 같은 그룹으로 분류

-Clustering 방법론 
Parametric 클러스터링 : 나누고자하는 클러스터의 개수 입력 받음
density-based 클러스터링 : 서로 근접하고 있는 샘플들 모두 한 덩어리
ensenble 클러스터링 : 여러 클러스터링 융합
Hieraechical 클러스터링 : 클러스터 크기가 작아질수록 클러스터 개수는 늘어남
	-> 최소한 크기로 클러스터 구성 후 그것들을 묶은 클러스터 형태를 단계별로 찾아냄


10-2 K-Means Clustering
-K-Means Clustering 정의
클러스터링이 가능한지 확인하기 위해 계속 사용됨
기본적으로 2개 input 필요 : 클러스터의 개수, Initial point(클러스터 가운데 점에 대한 포인터 값)

각 샘플들에 대해 가장 가까운 클러스터 중심지에 해당되는 인덱스를 클러스터 인덱스로 할당
	-> 클러스터 인덱스별로 묶인 샘플들의 평균 계산
	-> 중심점 위치 바뀌게 되고 클러스터 변화 없을 때까지 이를 반복

-K-Means Clustering 작동 방법
유클리디안 디스턴스 활동 (거리)
L1-norm : 노이즈 줄임

장점
	새로운 테스트 샘플 주어졌을때 바로 클러스터 인덱스 추정 가능
	모든 데이터 고려할 필요 없음 -> 무작위 추출

단점
	계산량이 큰 편
	매 반복마다 거리 비교
	평균으로 중심점 찾아야함
	처음 중심점이 잘못 선택된 경우 잘못된 결과 -> K-Means++ Clustering

-K-Means++ Clustering 정의
확률적으로 순서대로 클러스터 중심지 추출
랜덤으로 한개 중심지 ->  멀수록 무작위로 뽑히도록 선택
